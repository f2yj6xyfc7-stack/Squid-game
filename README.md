🎮 Squid Game Website

A web-based project inspired by the famous series Squid Game.
The website provides an interactive experience that takes the user through multiple pages, ending with a simple game.

⸻

📌 About the Project

This project is a multi-page website based on the theme of Squid Game.
It was designed to simulate a short journey for the user, starting from an introduction and moving through different sections until reaching the final game.

⸻

🚀 Pages Overview

* 🟥 Opening Page
    Displays an introductory image from Squid Game with a button that navigates to the next page.
* 📖 Summary Page
    Contains an image and a short description about the series, with a button to continue.
* 🔐 Login Page
    A simple login interface for user interaction.
* 👥 Main Characters Page
    Shows the main characters of the series.
    This page automatically redirects to the next page using JavaScript.
* 🎯 Red Light Green Light Game Page
    A simple interactive game inspired by the famous scene in Squid Game.

⸻

🛠️ Technologies Used

* JavaScript
